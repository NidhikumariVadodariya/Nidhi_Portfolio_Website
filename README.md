# Nidhi Vadodariya Portfolio Website

## Overview

This portfolio website showcases my web development projects, technical skills, education, and professional background. The website was designed and developed to demonstrate my ability to create responsive, user-friendly, and visually appealing web applications using modern front-end technologies.

## Live Demo

[Portfolio Website](https://nidhi-vadodariya-portfolio.netlify.app/)

## Features

* Responsive design for desktop, tablet, and mobile devices
* Sticky navigation with smooth scrolling
* Project showcase with screenshots and live demos
* Skills and education sections
* Resume download functionality
* Contact information and professional links
* Modern and clean user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
